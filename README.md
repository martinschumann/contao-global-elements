# contao-garage/contao-global-elements


Contao bundle supplying a backend module to help organizing numerously reused content elements like contact boxes, announcements, author boxes etc.

Content elements reused by referencing can be stored in site-independent archives.

For referenced elements in these archives the details pop up is replaced by an article tree to introspect and edit all references of an element (does not include usage in insert tags).

## Installation

Add a composer dependency for this bundle. Change in the project root and run:

```bash
composer require contao-garage/contao-global-elements
```
